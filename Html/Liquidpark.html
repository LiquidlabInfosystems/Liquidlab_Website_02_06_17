LiquidPark
Intelligent CarPark Guidance System

LiquidPark is an effective and economic solution for Carpark Guidance Systems.The system follows a zone based counting principle with induction loop sensors on the driveways.Our solid direction finding logic identifies the direction of vehicle movement, any tail gating events and many other irregular vehicle movement scenarios.
The System provides remote control, monitoring and reporting facilities through windows and Smartphone applications.The windows application uses detailed visual indicators to convey maximum information regarding the carpark status to the user in a single view. The smartphone application provides detailed realtime information and notifies the user using google clound messaging system in case of any events like "Carpark Full".
For better guidance, the system provides occupancy information at every strategic locations using variable message signs.LiquidPark uses IP based 4 digit displays with arrows.
Ultrasonic detectors along with the pilot lights on each carpark lot provides a better visual indication of the availability of the carpark lot to the drivers. Our detectors are one the smallest of its type in the market with 60mm diamter and 45mm height ,which makes it literally invisible inside the carpark.
